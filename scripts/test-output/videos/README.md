## 🔧 使用方法

1. 将 `videos.json` 复制到项目的 `lib/config/` 目录
2. 在网站配置中设置 `isShowVideo: true`
3. 确保所有视频链接可以正常访问

## ⚠️ 注意事项

- 支持 YouTube、Bilibili、Vimeo 等主流平台
- 确保视频内容与游戏相关
- 建议视频时长在 1-10 分钟之间
